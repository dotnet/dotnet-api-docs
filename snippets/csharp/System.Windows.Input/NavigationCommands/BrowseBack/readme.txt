This Project contains at least 3 different samples. 
To run the different samples, you will need to change the StartupUri in App.xaml to the root for
the different sample projects. If you figure out other samples in here, please add to this list.
-Zoom.xaml - demonstrates zoom in documents
-NavigateJournal - demonstrates many of the navigation types including NavigationCommands
-Search - demonstrates Search (broken, bug 819129)
