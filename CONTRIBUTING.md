# Contributing

Thank you for your interest in contributing to the .NET documentation!

This repository contains the API Reference .NET documentation. The main repository for .NET documentation is the [.NET Docs repository](https://github.com/dotnet/docs). See the [Contributing Guide](https://docs.microsoft.com/en-us/contribute/dotnet/dotnet-contribute) for information on updating both .NET Docs conceptual and .NET API reference documentation.
